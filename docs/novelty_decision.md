# Novelty Decision

Decision after v3 full-scale hardening: keep as final submission artifact.

The live novelty is not contact latency compensation. The live novelty is a robot-centered state variable for self-generated photometric evidence. Prior work covers cast-shadow scene reasoning, shadow removal, robustness to self-shadowing, and visual self-modeling. The v3 manuscript occupies the gap between them: robot-caused shadows and reflections are used as state for self-occluded hidden pose and clearance decisions.

The full-scale benchmark supports the novelty because non-shadow visual, kinematic-only, shadow-removal, and generic photometric baselines all remain below the robot self-shadow state, while the oracle remains above it.
